# Fullstack Financeiro

Aplicação **fullstack** para controle financeiro.

## Estrutura do projeto
- `backend` → API (Node.js)
- `frontend` → Aplicação mobile (React Native + Expo)

## Pré-requisitos
- Node.js
- npm ou yarn
- Expo CLI

## Como executar o projeto

### Backend
Abra um terminal:
```bash
cd backend
node server.js
